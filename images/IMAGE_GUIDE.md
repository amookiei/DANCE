## Figma에서 이미지 다운로드 방법

1. Figma에서 해당 이미지 레이어 선택
2. 오른쪽 패널 하단의 **Export** 섹션 확인
3. **+** 버튼 클릭하여 내보내기 설정 추가
4. 형식: **JPG** (사진) 또는 **PNG** (로고/아이콘) 또는 **SVG** (벡터 로고)
5. 배율: **2x** (고해상도 권장)
6. **Export** 버튼 클릭

## 이미지 최적화 팁

- JPG: 품질 80-85% 정도로 압축
- PNG: TinyPNG 등으로 압축
- WebP 형식 사용 시 더 작은 용량 가능
